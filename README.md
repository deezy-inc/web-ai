# @deezy-inc/web-ai

AI in the browser, powered by micropayments

## Installation

```bash
npm install @deezy-inc/web-ai # or yarn add @deezy-inc/web-ai
```

## Usage

NodeJs & Browser

```js
const webai = require('@deezy-inc/web-ai');
const { utils } = webai;

console.log(utils.normalizeURL('https://google.com'));
```
